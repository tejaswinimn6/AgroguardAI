# 🌿 AgroGuard AI

AgroGuard AI is an AI-powered Plant Disease Detection and Treatment Recommendation System.

## 📌 Project Description

This project helps detect plant diseases based on plant symptoms and provides treatment and prevention suggestions using AI.

## 🚀 Features

- Detect plant diseases
- Provide treatment suggestions
- Suggest prevention methods
- AI-powered using Google Gemini

## 🛠 Technologies Used

- Python
- Streamlit
- Google Gemini API

## ▶ How to Run the Project

1. Install required libraries:

pip install -r requirements.txt

2. Run the project:

streamlit run app.py
